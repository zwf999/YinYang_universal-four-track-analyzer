# 通用四轨道分析系统

**Universal Four-Track Analyzer**

## 项目简介

本系统是一个通用的四轨道分析工具，可用于分析多种类型的数据：
- **DNA序列**：分析DNA碱基对模式、GC含量、配对特征
- **数学常数**：分析π、e、黄金分割率等常数的数字特征
- **物理常数**：分析光速、普朗克常数等物理常数的数字特征

## 核心功能

### 🧬 DNA分析功能
- **四轨道分析**：同时分析DNA序列的四个维度特征
- **正反向分析**：完整的正向和反向序列分析
- **方向标记**：使用左箭头（←）标记反向碱基对
- **三角形编码**：将DNA碱基对编码为0-9数字
- **疾病预测**：基于特征差异进行癌症风险预测

### 🔢 常数分析功能
- **多轨道分析**：从不同维度分析常数的数字特征
- **模式识别**：识别数字序列中的重复模式和对称性
- **统计分析**：计算数字分布、配对率等统计特征
- **可视化输出**：生成详细的分析报告

### 🚀 技术特点
- **内存优化**：支持处理大型DNA序列和长数字串
- **批处理**：支持同时分析多个文件
- **详细报告**：自动生成comprehensive analysis reports
- **可扩展性**：易于添加新的分析方法和数据类型

## 安装与使用

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 分析DNA序列
```bash
# 运行通用分析器（会分析data目录中的所有文件）
python dna_universal_analyzer.py
```

### 3. 疾病预测分析
```bash
# 运行疾病预测器（基于分析结果）
python new_disease_predictor.py
```

### 4. 查看结果
- **分析结果**：`results/` 目录（JSON格式）
- **分析报告**：`reports/` 目录（文本格式）

## 编码原理

### DNA三角形编码表
```
(0, 0): 0    # AA
(0, 1): 1    # AC
(1, 1): 4    # CC
(0, 2): 2    # AG
(1, 2): 5    # CG
(2, 2): 7    # GG
(0, 3): 3    # AT
(1, 3): 6    # CT
(2, 3): 8    # GT
(3, 3): 9    # TT
```

### 方向标记
- **正序**：无标记
- **逆序**：左箭头（←）
- **单个碱基**：直接使用0-3表示

## 四轨道分析系统

### 轨道1：符号配对分析
- 基于数字属性的八卦配对规则
- 窗口滑动分析（12位窗口，步长5）

### 轨道2：和为9配对
- 数字和为9的配对分析
- A(1-8), B(2-7), C(3-6), D(4-5), E(9-0)

### 轨道3：和为10配对
- 数字和为10的配对分析
- 甲(1-9), 乙(2-8), 丙(3-7), 丁(4-6), 戊(5-0)

### 轨道4：特定组合配对
- 基于特定数字组合的配对分析
- 一(1-8), 二(2-5), 三(3-6), 四(4-7), 五(9-0)

## 疾病预测模型

### 关键特征
1. **未配对数** (total_unpaired)：癌症样本显著更高
2. **阴阳失衡** (yang_imbalance)：癌症样本显著更高
3. **GC含量** (gc_content)：癌症样本显著更低
4. **轨道4配对率** (track4_pairing_rate)：癌症样本显著更低

### 预测准确率
- **测试样本**：10个（5健康 + 5癌症）
- **准确率**：100.0%
- **最佳特征区分度**：43.0%
