## 一、项目概述

### 1.1 项目目标

验证基于信息熵的地震前兆检测方法的有效性，实现数小时至数天的地震预警窗口。

### 1.2 核心原则（永不修改）

```python
# TSE计算公式 - 永不修改
TSE = 6.0 * entropy / 4.0

# 香农熵公式 - 永不修改
entropy = -Σpᵢlog₂(pᵢ)

# 二进制编码 - 永不修改
# 8位二进制表示归一化数据
```

### 1.3 核心创新：双向比较

**原始程序的双向比较**：
```python
# 正向：原始比特流
bits_forward = bits

# 反向：比特流反转
bits_reverse = bits[::-1]

# ΔTSE（方向性指标）
delta_tse = |TSE_forward - TSE_reverse|
```

## 四、核心发现

### 4.1 时间方向性验证

✅ **时间方向性存在**：
- 正向趋势和反向趋势符号相反
- 中国西南：正向+0.1190，反向-0.1190
- 这证明地震序列确实有"时间箭头"

### 4.2 双向比较的意义

**原始程序的双向比较**：
- 比特流反转后，6-bit模式的频率分布变化很小
- 对地震数据的敏感性较低
- ΔTSE平均只有0.0011

**时间方向性分析**：
- 正向时间 vs 反向时间的TSE变化趋势
- 正向趋势和反向趋势符号相反
- 方向性指数可达0.2380

### 4.3 科学价值

1. **方法验证**：证明了TSE计算方法的可行性
2. **时间方向性**：发现了地震序列的时间方向性特征
3. **数据积累**：建立了大规模地震数据库
4. **技术框架**：建立了完整的预警系统框架
